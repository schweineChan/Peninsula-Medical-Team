# 维韧（WITHIN）打法交互课件

## 打开方式

双击 `index.html`，建议使用 Chrome 或 Edge 打开并点击右上角“全屏”。

## 操作

- `←` / `→` / 空格：翻页
- `M`：目录
- `F`：全屏
- `N`：讲者备注
- `T`：15 分钟授课计时
- 触屏设备支持左右滑动

## 内容结构

共 11 页，建议授课 15 分钟：

1. 封面与课程目标
2. 结构差异互动题
3. WITH + IN 概念
4. WITHIN 六步地图
5. Watch + Identify
6. Target
7. Hierarchy 参数推演
8. Integrate 决策拼图
9. Natural 结果选择
10. 一分钟快测
11. 总结

内容依据：《维韧打法简介-定稿》《半岛大超炮标准化实操-定稿（正文版）》。
